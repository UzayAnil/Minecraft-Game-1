Game Instructions

Please press the "New Game" button to start a new game. The goal of the game is to use the given tools to move the land and landscape. The player has access to three tools: a pickaxe for the rocks; an axe for the bush and tree; a shovel for the grass and dirt. The player must select a tool to grab the item, then re-click the tool to put it back.
